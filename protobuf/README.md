# Protobuf

## Formatter

Format with `prototool format`.

## Linter

Lint with `prototool lint`.

Configure `prototool` to use the `uber2` linter.

```yaml
lint:
  group: uber2
```

## Style guide

Look to [Uber's Protobuf Style Guide V2][uberv2].

[uberv2]: https://github.com/uber/prototool/tree/dev/style
